Takes a namespace string and fetches that location out from an object. If the namespace doesn't exists it is created then returned.

Parameters:

  • ns_string String

    the namespace you wish to fetch

  • extendObj Object

    optional, default this.options

  • valueToSet Object

    optional, a value to set on the key (last key if key string (a.b.c.d = value))

Returns:

Object:

Example:

_namespace('hello.world.how.are.you.doing') // will create/fetch:

{hello: { world: { how: { are: { you: { doing: {} } } } } } }
